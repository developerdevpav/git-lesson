Два состояния файлов в git: отслеживаемые и неотслеживаемые.
Статусы файла: untracked (неотслеживаемые), unmodified (не изменённые), modified (изменённые), staged (готовые к commit).

git status - определяет состояние файлов на текущий момент.

git add <file> - добавить в отслеживаемые.

git diff - подробная информация о состояниях файла (ов).
